Este proyecto tiene como objetivo desarrollar una solución basada en Machine Learning que permita predecir cuándo un producto podría quedarse sin stock en una empresa de retail.

La problemática surge debido a la dificultad de anticipar la demanda, lo que provoca quiebres de stock, pérdidas económicas y baja satisfacción de los clientes.

A través del análisis de datos históricos, se busca generar predicciones que ayuden a mejorar la toma de decisiones en la gestión de inventario.
